About

Eggenda keeps you productive and entertained! More than just a to-do list app, it rewards your accomplishments with points to level up and unlock characters, pets, and accessories. This fun reward system motivates you to complete tasks quickly and stay engaged.

Why Eggenda ?

This is the frist app introducing todolist - game app to users.

The very first app that can make WORK-LIFE BALANCE, if you are tired to finish the task, go play some fun games to relax!

When you do task for a long time, go for a walk,this could help you hetch an egg too! Do not just sit infront of the desk!

Show and Tell 1

In Show and Tell 1, we were able to implement a good foundation of Eggenda for future additional features.

  • Home page: Developed the basic logic behind our final product for the egg hatching and quest board.
  • Settings: Built user profile settings and change the language of the app.
  • Game: Implement a battle system that is easy for the user to understand. The game also needs the user to develop strategies and presents difficulties, allowing for a challenging but fun battle.

Show and Tell 2

In Show and Tell 2, we were able to implement the app's databases, game implementation, tasks/quests.

  • Home page: Implement adding/editing/deleting tasks and quests along with countdown timers and checkbox features
  • Game: NEED HELP HERE, Implement gallery page UI, along with pet selection, monster stages.
  • Task entry database: Store tasks entered by user to room database to keep track of all tasks
  • User database: Store users in room database and firebase database to sync user data between devices
  • Login/Create account: Allow users to login to a previously created account or create a new account to the database
